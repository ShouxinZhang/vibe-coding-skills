# CNN：从看局部到看整体

> 本模块不是先堆公式，而是先回答三个更重要的问题：为什么 CNN 会出现，它到底比全连接网络自然在哪里，以及它是怎样一步步长成视觉领域基础设施的。

---

## 模块定位

卷积神经网络（Convolutional Neural Network, CNN）是深度学习里最早大规模改变产业的模型家族之一。它的重要性不只在于图像分类拿了多少冠军，而在于它第一次把一个很朴素的想法做成了可训练系统：**数据如果本身有结构，模型就应该尊重这种结构。**

图像不是一串毫无关系的数字。相邻像素通常彼此相关，同一种纹理会在不同位置反复出现，复杂物体又是由简单边缘和局部形状一步步组合起来的。CNN 的成功，本质上就是把这些直觉写进了网络结构里。

这一组文章会更强调背景、来龙去脉和直觉演化，而不是一开始就把所有数学细节压给读者。

## 一、为什么 CNN 值得单独学

CNN 不只是“图像版神经网络”。它代表了一种更通用的工程思想：**让模型利用任务本身的先验，而不是只靠堆参数硬学。**

这件事的意义很大，因为很多后来你会见到的结构设计，本质上都在重复这条路线：

1. 先找出数据最稳定的规律。
2. 再把这些规律编码进结构。
3. 用更少参数换来更强泛化。

所以学习 CNN，不只是学习一个老模型，而是在学习“好架构是怎么长出来的”。

## 二、CNN 的历史位置

CNN 很早就被提出，但真正爆发是在 2012 年 AlexNet 之后。原因并不神秘：方向早就对了，只是当时缺少足够大的数据集、足够强的 GPU 和足够稳定的训练技巧。

如果把 CNN 的历史压缩成一条主线，大致可以这样看：

| 阶段 | 核心问题 | 代表结果 |
|------|----------|----------|
| LeNet 时代 | 证明卷积思路可行 | 卷积 + 池化 + 分类的基础范式建立 |
| AlexNet 时代 | 让深度视觉模型真正打败传统方法 | 深度学习成为视觉主流 |
| VGG 时代 | 证明“小卷积核 + 更深结构”有效 | 架构变得规整、可迁移 |
| ResNet 时代 | 解决“更深但训不动” | 深层网络真正可规模化使用 |
| EfficientNet 时代 | 更系统地平衡精度与成本 | 缩放方法论成熟 |

你会发现，每一代架构都不是凭空出现，而是在修补上一代暴露出来的问题。

## 三、这一模块主要帮你建立什么直觉

### 1. 背景直觉

为什么图像任务会逼着研究者走向 CNN，而不是继续用普通全连接网络。

### 2. 结构直觉

卷积核、特征图、感受野、步长、池化这些词，各自在模型里扮演什么角色。

### 3. 演化直觉

LeNet、AlexNet、VGG、ResNet、EfficientNet 不只是模型名单，而是一条连续的改进链。

### 4. 现实直觉

CNN 现在仍然有价值，但它也有边界。它擅长局部结构明显的任务，不天然擅长超强全局依赖。

## 五、小结

CNN 的成功并不是偶然，它抓住了图像最重要的三个事实：

1. 图像里的规律通常先发生在局部。
2. 同一种局部模式会在不同位置重复出现。
3. 高层语义是由低层模式逐步组合出来的。

理解这三点，再去看卷积公式、模块设计和架构演化，很多看似分散的知识就会自然串起来。
