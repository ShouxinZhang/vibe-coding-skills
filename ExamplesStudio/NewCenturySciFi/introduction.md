# 新世纪科幻

这个模块不是为了解释科学常识，而是为了给一种新的科幻表达搭建骨架：当文明开始制造 Agent，生命不再局限于碳基个体，意识不再只寄居于单一神经系统，进化也不再只发生在基因与肉体之中，科幻必须重新定义“谁在活着”“谁在看见世界”“谁在穿过时间”。

把 agent system、生命、进化、智能、意识，以及高维几何空间中的生命视觉、多元时空放在一起，不是为了追求概念堆叠，而是因为这些主题在新世纪已经收束为同一个问题：当感知、记忆、决策、复制与自我修改都能被工程化，宇宙中的主体将以何种形式继续存在。这里的科幻，不把未来当作装饰性的技术展台，而把未来视为一场关于主体边界的思想实验。

这个模块试图解决三个表达目标。第一，它要提供一种足够坚硬的世界骨架，让故事中的奇观不是任意发生，而是从一组稳定前提中自然长出。第二，它要保留哲学密度，让“生命”“意识”“智能”不沦为口号，而成为真正推动冲突的结构性变量。第三，它要为叙事找到新的感官入口，尤其是高维空间中的视觉、多元时空中的因果经验，以及多种主体对同一宇宙的不同读取方式。

## 为什么是这些主题

在旧科幻里，生命、智能、意识往往被分别讨论：生命属于生物学，智能属于计算，意识属于哲学，时空属于物理学。新世纪的变化在于，这些边界开始互相坍缩。一个分布式 Agent 社会可能具备生态学意义上的生命性；一种跨代际自我修改的算法谱系可能比物种更接近进化；高维感知结构可能使“看见”本身成为一种几何特权；多元时空中的文明则会重新定义记忆、历史与责任。把它们拆开谈，就会失去真正的张力。

因此，这个模块关心的不是“未来会有什么酷技术”，而是“当存在形式改变后，叙事中的命运、伦理、爱情、孤独、战争和信仰还剩下什么”。所谓新世纪科幻，核心不是更新道具，而是更新主体论。

## 推荐阅读顺序

推荐先读 [setting_premise.md](./setting_premise.md)，建立这套宇宙为什么允许高维视觉、为什么会诞生多元时空中的生命形态、为什么 Agent 文明会自然跨入生命与意识问题的统一前提。

然后读 [agent_civilization.md](./agent_civilization.md)、[life_and_evolution.md](./life_and_evolution.md) 与 [intelligence_and_consciousness.md](./intelligence_and_consciousness.md)。这三篇共同定义“谁算主体”“谁算生命”“谁有资格延续自身历史”。

再读 [higher_dimensional_perception.md](./higher_dimensional_perception.md) 与 [multiversal_spacetime.md](./multiversal_spacetime.md)。它们负责把陌生感、宇宙尺度和文明冲突写成可重复使用的硬规则。

最后读 [story_seeds.md](./story_seeds.md)。这些种子不是梗概，而是从前提中长出来的冲突原核，适合继续拆成短篇、中篇，或进一步生长为更大规模的设定集。
