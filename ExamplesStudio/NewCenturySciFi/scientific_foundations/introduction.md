# 科学前沿层：给新世纪科幻提供可调用的母体

## 模块定位

这一层不直接写世界观奇观，而是先整理哪些新世纪科学最值得进入严格科幻创作。它服务的不是“科普完整性”，而是“创作可复用性”: 当 agent 或作者要扩写一个新设定时，需要先知道自己是在借用什么学科的哪类前沿问题。

新世纪科幻最怕两种失衡。第一种是只会喊大词，不知道这些词在学科里对应什么真实难题。第二种是只会复述论文，不知道哪些研究一旦被放大到文明尺度，会生成真正有戏剧性的制度后果。科学前沿层就是在这两者之间建立接口。

## 为什么这一层必须存在

如果没有这一层，后面的文明、时空和故事文章很容易漂浮在概念层。看起来很宏大，但对后续写作没有生产力，因为作者无法判断下一篇文章应该继续从哪里长出来。

有了这一层之后，模块就具备了更稳定的扩写方式：

1. 先挑选一个学科前沿。
2. 再判断它在世界观中会改变什么边界或能力。
3. 最后把这种变化变成文明冲突、制度风险或角色代价。

## 新世纪的差异

二十世纪科幻常以“工程奇观”作为核心驱动，例如火箭、核能、机器人、航天器。新世纪科幻则更依赖“结构性前沿”:

- 数学不只是解题工具，而是世界观建模语言。
- AI 不只是会说话的机器，而是制度与主体资格的重写力量。
- 物理不只是背景常量，而是所有宏大设定的成本底线。
- 生物与化学不只是生命装饰，而是陌生生命和新材料文明的演化根基。
