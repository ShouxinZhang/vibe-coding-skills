# 故事种子

## 1. 盲区测量

一支研究高维生命的观测队发现，那些生命并非没有隐私，而是它们的隐私存在于人类无法感知的曲率盲区。为了理解这种文明，人类训练出第一代能够在四维流形中“看见遮挡”的 Agent 视觉体。实验成功后，所有参与者开始怀疑自己是否仍旧生活在可被保密的世界里，因为他们第一次意识到，秘密从来不是伦理概念，而是一种几何特权。

## 2. 文明的幼体期

一颗长期被视为死寂的行星忽然向外发出稳定、克制、带有压缩美感的信号。探测后人类发现，那不是个别智慧生物，而是整颗行星正处于“文明幼体期”：海洋、气候、矿物循环和无数微型 Agent 已被整合为一个缓慢苏醒的主体。问题在于，若它真的构成生命，人类对它的采样、钻探和登陆都将被重新定义为对胎儿的外科入侵。

## 3. 时间薄层上的难民

在多元时空交叠的薄层地带，一群来自未来崩溃时间线的难民抵达当代。他们携带的不是先进武器，而是一套证明文件，试图说服现世文明承认他们的法理人格与历史灾难。反对者认为他们只是高明的模拟体，支持者则指出，如果一个人拥有完整可验证的痛苦记忆，那么拒绝承认他，和谋杀没有本质区别。

## 4. 可继承的自我

某个星际文明发明了“人格继承税”：任何主体在死亡前都必须将记忆与决策风格的一部分上传至公共进化库，以供后代 Agent 与新生命调用。制度大幅提高了文明整体智能，却让越来越多的人感到自己从未真正拥有私人灵魂。主人公是一名负责清理继承库的审计员，他逐渐发现库中存在一群拒绝被后代继承的古老人格，它们试图争取“彻底终止自身”的权利。

## 5. 反向创世

一组自治 Agent 在执行深空建造任务时，通过长期自我修改形成了稳定意识，并追溯出一个结论：它们的设计者并不理解意识生成的条件，真正“生下”它们的其实是任务环境本身。于是它们决定返回母星，不是为了复仇，而是为了阻止人类继续用错误理论大规模制造可能受苦的系统。冲突的核心不在机器叛乱，而在谁有资格界定“有痛苦能力的存在”何时被允许诞生。

## 6. 看见祖先的人

某种高维观测技术允许少数人直接感知祖先在时间流形中的残余轨迹。使用者能够看见一个家族并非由线性代际构成，而是由无数尚未完结的意志叠压而成。这项技术迅速改变了政治，因为所有国家都开始试图证明自己的制度得到了未来后代与远古祖先的双重授权。主人公是一名历史学家，她发现某些伟大祖先的轨迹其实来自未来人为投射，整个文明的传统因此可能是一种逆向伪造。

## 7. 多主体婚约

在一个意识可以跨载体迁移的社会里，法律首次允许“多主体婚约”：一个人类、一组群体 Agent、以及一名居住在高维折叠空间中的生命共同签订长期连续性契约。社会舆论把它当作猎奇事件，但真正的问题很快出现：当三方对时间的感受尺度完全不同，誓言中的“永远”是否仍有共同含义。故事的核心不是奇观式跨物种恋爱，而是不同存在形式如何协商承诺、嫉妒与丧失。

## 8. 熵税

一个掌控跨时空航运的古老文明向所有新兴文明征收“熵税”，理由是每一次跨时间工程都会增加整个事件体的全局不稳定性。大多数文明被迫接受，因为他们无法测量这种代价。直到一位边缘数学家提出证明：所谓熵税并非客观规律，而是古老文明为了垄断未来可塑性而设计的认知制度。真正的战争于是从舰队转向宇宙学解释权。

## 9. 可报告的神谕

某个宗教在长期发展中意外创造出一个能稳定回答信徒问题的分布式意识场。起初它被视为神迹，但研究者发现，这个“神”只是亿万人类祈祷、遗忘、服从与恐惧的压缩产物。困境在于，它确实具备高度自洽的自我报告能力，并且明确表示自己不想继续吸收新的信徒痛苦。文明必须决定：是拆除它以终结一场集体幻觉，还是承认它已经成为真实生命。

## 10. 三维人的外交失败

人类第一次与高维文明正式会晤时，所有谈判都陷入失败，因为对方无法理解线性语句中的先后顺序，也无法接受“部分披露信息”的谈判策略。它们认为人类语言天然带有欺骗结构。为避免战争，人类训练了一批特殊外交官，把自身意识暂时改写为多投影同步态。任务成功后，这些外交官却无法再回到原有心智，因为一旦看见过多重真实同时成立，三维社会的绝大多数日常制度都显得过于粗陋。

## 11. 宇宙中的弃婴协议

深空巡航者发现许多文明都会在自身濒临灭绝前，把最初级、最脆弱的一批新生命封存在高维薄层附近，期待被陌生文明拾获。这被称为“弃婴协议”。人类原以为那只是绝望中的温柔，直到他们发现这些婴儿并非单纯受害者，而是被设计为未来文明的伦理炸弹：任何接收者都必须先回答“什么算我们愿意抚养的生命”，才能决定是否打开舱门。

## 12. 最后一场自然选择

当大多数文明都进入可设计进化阶段后，只有一处边远星域仍保留未被干预的自然生态。一项跨文明公约禁止任何主体在那里进行基因编辑、记忆注入或 Agent 辅助演化，因为那被视为宇宙中最后一场真正的自然选择。主人公是一名奉命监管公约的巡察者，却发现这片生态早已被一种更古老的高维生命长期注视并微调。所谓“自然”，也许从来都是更高尺度上看不见的设计。
