# vibe-coding-skills

[English Home](README.md) | [中文版本](README.zh-CN.md)

面向“氛围式高效编码”的知识与实践仓库，聚焦可复用的方法、范式与案例，帮助团队在有限时间内产出可验证价值。

## 我们的核心价值
我们提供一套可复制的工作方法，把“想法”快速转化为“可交付成果”，以更低的沟通成本、更快的试错速度，持续稳定地把业务目标落地。

## 我们的核心竞争力
- **结构化产出能力**：将思路沉淀为模块化知识与案例，便于团队复用与规模化传播。
- **高效迭代能力**：围绕真实问题组织内容，强调验证与反馈闭环，提升交付效率。
- **跨场景迁移能力**：用统一方法论覆盖不同业务场景，减少学习成本，提升整体协作效率。
- **富例驱动的学习体验**：含有丰富具体例子，易于用户快速学习理解。

## 仓库结构
- BasicKnowledge/：基础知识与参考
- ExamplesStudio/：小型示例与实验
- MyThought/：个人思考与总结
- Problems/：问题清单与记录
- docs/architecture/：仓库架构文档

## 使用方式
- 直接阅读各模块内容，理解方法与案例
- 结合业务场景，复用结构化模板与实践经验

## 贡献方式
- 新内容放在最合适的叶子模块目录
- 保持模块隔离与结构化输出
- 变更前查看并遵守 docs/architecture/repository-structure.md
