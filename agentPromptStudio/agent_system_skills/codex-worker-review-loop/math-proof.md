Let

$$
S_n = 1 + 3 + 5 + \cdots + (2n-1).
$$

We prove by induction that $S_n=n^2$ for all $n\ge 1$.

Base case: $S_1=1=1^2$.

Inductive step: assume $S_n=n^2$. Then

$$
S_{n+1}=S_n+(2n+1)=n^2+(2n+1)=(n+1)^2.
$$

So the statement holds for $n+1$. Therefore, by induction,

$$
1+3+5+\cdots+(2n-1)=n^2
$$

for all $n\ge 1$.
